About LatinCoach

LatinCoach is a single-page web application designed to help GCSE students revise Latin literature, grammar, and vocabulary for OCR and Eduqas exam boards.

The app features Quizlet-style literature flashcards, a vocabulary typing trainer, grammar reference tables with practice mode, and timed tests with strict marking.

All progress, scores, and favourites are saved locally in your browser per user account.

Data storage: All progress, scores, and favourites are saved to a private GitHub Gist on your account. You can access your data from any device by logging in with the same GitHub Personal Access Token. Your token is saved locally in this browser for auto-login.

Built with Tailwind CSS. Classical Roman theme with dark mode support.

Help

Vocabulary Typing Trainer: You are shown an English meaning and metadata. Type the Latin headword exactly (macrons are optional). Press Enter or click Check. Use Reveal if stuck. Navigate with arrow buttons.

Literature Flashcards: Tap or click the card to flip between Latin (front) and English (back). Use Prev/Next to navigate. Below the card is a list view of all flashcards. Use "Copy for Quizlet" to export in tab-separated format.

Grammar Practice: You are given a noun and asked to provide a specific case/number form. Macrons are optional. Strict marking.

Tests: 10 questions per test (declension or conjugation). Your score is saved. Review mistakes on the Progress page.

Login: Uses a GitHub Personal Access Token (gist scope). Your data is saved to a private GitHub Gist and accessible from any device with the same token. Your token is cached in this browser for auto-login.
